### Cross-project candidates file (2026-05-27)
`~/.claude/memory/.cross-project-candidates.md` is auto-generated by `check-cross-project.py` on every SessionStart.
During Step 1 of distill — after reading MEMORY.md — check if this file exists and load it.
Any slug listed there is a pre-flagged promotion candidate (same `name:` found in ≥2 project memories).
Use it to pre-populate the promotion candidates table; still verify the entry bodies match before promoting.
Delete the file after distill completes (it regenerates next session).
